Tentang Saya

Saya adalah seorang pengembang perangkat lunak yang bersemangat dan ingin terus belajar untuk mendapat pengalaman dalam pengembangan aplikasi web dan mobile. Saya memiliki latar belakang pendidikan di bidang ilmu komputer dan selalu bersemangat untuk mempelajari teknologi baru. Saya memiliki pemahaman dasar serta bisa mengimplementasikan beberapa bahasa pemrograman seperti Php, JavaScript, kotlin,java, Python, dan java serta bisa menggunakan kerangka kerja seperti Laravel dan Codeigniter. Selain itu, saya juga memiliki keterampilan dalam pengelolaan basis data. Saya senang bekerja dalam tim dan selalu mencari cara untuk meningkatkan diri saya dalam setiap proyek yang saya jalani.
